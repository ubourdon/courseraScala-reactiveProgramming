# Monads (20:22)

Une monad M est un type parametrisŽ M[T] avec 2 operations, flatMap & unit
qui doivent satisfaire plusieurs lois

trait M[T] {
    def flatMap[U](f: T => M[U]): M[U]
}

def unit[T](x: T): M[T]

Dans la literature, flatMap est communŽment appelŽ 'bind'

Exemples of monad :
    List is a monad awith unit(x) = List(x)
    Set is a monad with unit(x) = Set(x)
    Option is a monad with unit(x) = Some(x)
    Generator is a monad with unit(x) = single(x)

# Monad Laws            

Une monad doit satisfaire 3 lois
    
    Associativity :
        (m flatMap f) flatMap g == m flatMap (x => f(x) flatMap g)

    Left unit :
        unit(x) flatMap f == f(x)

    Right unit
        m flatMap unit == m

Monoid respectent la premire loi (ex: Int)

# check monads law with Option

abstract class Option[+T] {
    def flatMap[U](f: T => Option[U]): Option[U] = this match {
        case Some(x) => f(x)
        case None => None
    }
}

    1) left unit law        OK
    
    Need to show: Some(x) flatMap f == f(x)
                  unit(x)
     
            Some(x) flatMap f              
    ==      Some(x) match {
                case Some(x) => f(x)
                case None => None
            }

    ==      f(x)
    
    2) Right unit law       OK
    
    Need to show: opt flatMap Some == opt
    
            opt flatMap Some
            
    ==      opt match {
                case Some(x) => Some(x)
                case None => None
            }

    == opt

    3) associative law      OK
    
    Need to show: (opt flatMap f) flatMap g == opt flatMap (x=>f(x) flatMap g)
    
            (opt flatMap f) flatMap g
    
    ==      opt match { case Some(x) => f(x) case None => None}
                match { case Some(y) => g(y) case None => None}
                
    ==      opt match {
                case Some(x) =>
                    f(x) match { case Some(y) => g(y) case None => None }
                
                case None =>
                    None match { case Some(y) => g(y) case None => None}
            }

    ==      opt match {
                case Some(x) =>
                    f(x) match { case Some(y) => g(y) case None => None }
                case None => None
            }

    ==      opt match {
                case Some(x) => f(x) flatMap g
                case None => None
            }
    
    ==      opt flatMap (x => f(x) flatMap g)

=> Option is a monad
